# Hacknet Extension 教程

> [!WARNING]
> 目前处于开发早期阶段，未正式投入使用。如果你需要教程，请暂时移步 https://github.com/FBIKdot/Hacknet-Extension-Tutorial

Hacknet Extension 是 Hacknet 的玩家扩展内容，由媒体与特定格式的文本（主要是 XML）组成。

学习 Hacknet Extension 的所需知识不多，只需要你熟悉 XML 基本语法与大概知道 Hacknet 的游戏内容。

教程将会假设你已经熟悉 XML 基本语法与 Hacknet 游戏内容。如果需要，你可以去 [Runoob](https://www.runoob.com/xml/xml-tutorial.html) 学习 XML 基本语法。
